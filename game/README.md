# 🎮 SUPER MARON WORLD 〜まろんの大冒険〜

愛犬「まろん」が主人公の、スーパーマリオブラザーズ風・横スクロールアクションゲームです。
画像・音声ファイルを一切使わず、グラフィック (ドット絵) も BGM・効果音も
すべて JavaScript のコードだけで生成しています。

## 遊び方

### かんたん: 単一ファイル版 (ダブルクリックでOK)

**`game/play.html`** をブラウザで開くだけで遊べます。すべてのコードを1つの
HTML に埋め込んであるので、ファイルをダウンロードしてダブルクリックするだけで動きます。

### 開発版 (モジュール構成)

`game/index.html` は ES Modules 構成のため、`file://` で直接開くと
ブラウザのセキュリティ制限で読み込めません (画面が真っ青のまま)。
簡易サーバー経由で開いてください:

```bash
cd maron_official
python3 -m http.server 8000
# → http://localhost:8000/game/ を開く
```

> `play.html` は `node game/tools/build-standalone.mjs` で
> モジュール構成から自動生成しています。

## 操作方法

| 操作 | キー |
|---|---|
| 移動 | ← → / A・D |
| ジャンプ (長押しで高く) | Z / SPACE / ↑ / W |
| ダッシュ・ファイア | X / SHIFT |
| スタート / ポーズ | ENTER |
| サウンド ON/OFF | M |
| ステージ選択 (タイトル画面) | 1〜4 (4 = ボス戦) |

スマホなど鍵盤がない場合は、URL に `?stage=4` を付けるとそのステージから始められます
(例: `play.html?stage=4` でいきなりボス戦)。

スマホ・タブレットでは画面上のタッチボタンで操作できます。

## ゲーム内容

- **全4ステージ**: オフィス (1-1) → 地下洞窟 (1-2) → 星ふる夜の空中ステージ (1-3) → 屋上のボス戦 (1-4)
- **ボス戦**: ボブヘアの女の子「ゆいちゃん」。追いかけ・ジャンプ・ハート投げの3攻撃。3回踏むと「なかなおり」できる
- **パワーアップ**: キノコで大きくなり、フラワーでファイアまろんに変身してファイアボールを発射
- **敵キャラ**: きいろちゃん (ぴょんぴょん跳ねる金色ハムスター)、おすしちゃん (ちょこちょこ走る白ハムスター)、ルルちゃん (空中を縦横無尽に飛び回る子犬)
- **ハテナブロック / レンガ**: コイン・パワーアップ・1UP が出現。大きいまろんはレンガを破壊できる
- **クラシックな仕掛け**: 旗ゴール (高い位置ほど高得点)、コイン100枚で1UP、制限時間、タイムボーナス
- **ハイスコア保存** (localStorage)

## 技術構成

- 素の JavaScript (ES Modules) + Canvas 2D。ビルド不要・依存ライブラリゼロ
- 固定タイムステップ (120Hz) の物理更新 + 可変フレーム描画
- コヨーテタイム・ジャンプ先行入力・可変ジャンプ高など、現代的な操作感の調整入り
- Web Audio API によるファミコン風のシンセ効果音と自作 BGM のステップシーケンサー

```
game/
├── index.html            エントリーポイント
├── css/style.css         画面レイアウト・タッチUI
└── js/
    ├── main.js           ゲーム本体 (状態遷移・進行・描画)
    ├── input.js          キーボード / タッチ入力
    ├── audio.js          効果音と BGM (Web Audio)
    ├── sprites.js        ドット絵とタイル・背景の描画
    ├── level.js          タイルマップ・衝突判定・レベルビルダー
    ├── camera.js         スクロールカメラ
    ├── hud.js            スコア表示
    ├── entities/         プレイヤー・敵・アイテム・パーティクル
    └── levels/           各ステージのデータ
```
